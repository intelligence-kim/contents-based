# contents-based 넷플릭스 작품 추천 시스템

## 프로젝트 소개
작품 리뷰를 수집하여 각 작품의 리뷰와 가장 비슷한 작품 10개를 추천해준다.

## 프로젝트 기술 소개
 - Tockenizer : nltk
 - 크롤링 라이브러리 : BeautifulSoup4, Selenium
 - 시각화 라이브러리 : wordcloud
 - 토픽 모델링 : LDA

## 개발 기간

## 구성원
 - 김지성

## 사용 언어
 - python
